<h1 align="center">TraceNet: Real-Time Network Tester</h1>

<p align="center">
Internet Speed Checker is a lightweight and efficient web-based tool designed to measure your network performance accurately. 
With a clean, minimalist UI, it provides instant insights into your download, upload, and latency speeds.
</p>

<p align="center">
<strong>TraceNet Live:</strong> 
<a href="https://huggingface.co/spaces/im-plabonn/TraceNet" target="_blank">
https://huggingface.co/spaces/im-plabonn/TraceNet
</a>
</p>

<hr>

<h2>⚙️ Tech Stack</h2>

<ul>
<li><strong>Programming Language:</strong> Python 3.x</li>
<li><strong>Core Framework:</strong> Gradio (for rapid and interactive UI/UX development)</li>
<li><strong>Testing Engine:</strong> speedtest-cli</li>
<li><strong>UI Styling:</strong> Custom CSS for responsive, centered alignment</li>
</ul>

<hr>

<h2>🚀 Key Features</h2>

<ul>
<li><strong>Real-time Diagnostics:</strong> Instantly measures download speed (Mbps), upload speed (Mbps), and ping (ms).</li>
<li><strong>Intuitive Dashboard:</strong> A clean, clutter-free interface that displays results clearly.</li>
<li><strong>Smart UI Design:</strong> Custom-styled components ensure all metrics are perfectly centered and easy to read.</li>
<li><strong>Timestamp Logging:</strong> Keeps track of the exact time of your latest network performance test.</li>
</ul>

<hr>

<h2>🧠 How It Works</h2>

<ol>
<li><strong>Initiate Test:</strong> Simply click the <em>"Run Speed Test"</em> button.</li>
<li><strong>Data Acquisition:</strong> The application connects to the nearest high-performance server to perform a standard speed test.</li>
<li><strong>Display Results:</strong> The dashboard updates dynamically to show your current network performance metrics.</li>
</ol>

<hr>

<h2>🎯 Purpose & Goal</h2>

<p>This project is built to demonstrate:</p>

<ul>
<li><strong>Efficient Backend Integration:</strong> Handling asynchronous network requests seamlessly within a Python-based web app.</li>
<li><strong>UI/UX Optimization:</strong> Leveraging Gradio's layout components combined with custom CSS to create a professional user interface.</li>
<li><strong>Performance Monitoring:</strong> Providing a simple, accessible way for users to monitor their internet connection stability.</li>
</ul>
