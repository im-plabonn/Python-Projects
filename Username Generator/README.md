<h1>Username Generator</h1>

<p>A simple yet efficient Python tool that generates creative and professional username suggestions based on a user's first and last name.</p>

<p><strong>Live Project:</strong> <a href="https://www.plabonrahman.me" target="_blank">www.plabonrahman.me</a></p>

<h2>Key Features:</h2>
<ul>
  <li><strong>Custom Suggestions:</strong> Generates 10 unique username variations (e.g., first.last, im-first, first123) based on your input.</li>
  <li><strong>Instant Copy:</strong> Each username comes with a convenient "Copy" button for quick and easy use.</li>
</ul>

<h2>Technologies Used:</h2>
<ul>
  <li><strong>Programming Language:</strong> Python</li>
  <li><strong>Interface Library:</strong> Gradio</li>
</ul>

<h2>How to Use:</h2>
<ol>
  <li>Enter your First Name and Last Name into the input fields.</li>
  <li>Click the "Generate Usernames" button.</li>
  <li>Browse the results and click the "Copy" button next to your preferred username to save it to your clipboard.</li>
</ol>
