<h1>Number Quest: The Ultimate Number Guessing Challenge</h1>
<h2>Some of the cool features will be added soon!</h2>

<p>
<b>Number Quest</b> is a lightweight, interactive, and highly addictive web-based number guessing game. 
Designed to test your logic and intuition, this project provides a seamless user experience with 
dynamic difficulty levels and social media integration.
</p>

<p>
<b>Live Demo:</b> 
<a href="https://huggingface.co/spaces/im-plabonn/Number-Quest">
Number Quest - Hugging Face
</a>
</p>

<h2>Tech Stack: </h2>

<ul>
<li><b>Programming Language:</b> Python 3.x</li>
<li><b>Core Framework:</b> Gradio (for rapid UI/UX development)</li>
<li><b>Web Integration:</b> HTML/JS injection for social sharing functionalities</li>
<li><b>Library:</b> urllib.parse (for dynamic URL encoding)</li>
</ul>

<h2>Key Features: </h2>

<ul>
<li><b>Dynamic Difficulty:</b> Choose between Easy (1-5), Medium (1-10), or Hard (1-15) modes.</li>
<li><b>Intuitive UI:</b> Clean, responsive, and user-friendly interface powered by Gradio's Blocks API.</li>
<li><b>Psychological Engagement:</b> Gamified result tracking with accuracy percentages to encourage competition.</li>
<li><b>Social Sharing:</b> One-click Facebook integration that allows users to share their scores directly to their timeline with a pre-filled draft text.</li>
<li><b>State Management:</b> Robust handling of game states (score tracking, attempt limiting, and visibility toggling) using session state.</li>
</ul>

<h2>How It Works: </h2>

<ul>
<li><b>Initialization:</b> The user selects a difficulty mode, triggering the backend to set the range and attempts.</li>
<li><b>Game Logic:</b> The system processes user inputs, validates them, and updates the score and attempts in real-time.</li>
<li><b>Completion & Sharing:</b> Once all attempts are exhausted, the system displays the final accuracy and renders a specialized <b>"Post to Facebook"</b> button.</li>
<li><b>Social Amplification:</b> By clicking the share button, users are prompted with a curiosity-driven message designed to increase viral engagement.</li>
</ul>
