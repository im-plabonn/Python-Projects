# Text Analyzer Tool

A clean, fast, and user-friendly web application built to analyze text content instantly. Whether you need to count words, characters, or identify case distribution, this tool provides real-time insights with a modern UI.

## Live Demo
Check out the live application here: [https://huggingface.co/spaces/im-plabonn/Word-Counter](https://huggingface.co/spaces/im-plabonn/Word-Counter)

## Key Features
The Text Analyzer provides an instant breakdown of your input:
- **Word Count:** Calculates the total number of words in the text.
- **Character Count:** Measures the total length of the content (including spaces).
- **Case Analysis:** Identifies the total count of uppercase and lowercase letters.
- **Space Counting:** Tracks the number of whitespace characters.

## Tech Stack
- **Programming Language:** Python
- **Frontend/UI Library:** [Gradio](https://www.gradio.app/)
- **Hosting:** Hugging Face Spaces

## How It Was Created
1. **Logic Building:** The backend logic was developed using Python's efficient string processing methods to ensure fast results even for large blocks of text.
2. **Interface Design:** I used the `Gradio Blocks` API to create a custom, responsive layout.
3. **Deployment:** The application was containerized and hosted on the Hugging Face platform, making it accessible from any device.

## Usage
1. Open the [Live Link](https://huggingface.co/spaces/im-plabonn/Word-Counter).
2. Paste or type any text into the **Input Text** box.
3. Click the **"Analyze Text"** button.
4. View the results instantly in the summary dashboard below.
